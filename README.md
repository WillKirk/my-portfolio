# willkirkpatrick.dev

My personal portfolio site, built to showcase my projects and experience.

## Tech stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Deployment

Deployed on Vercel at [willkirkpatrick.dev](https://willkirkpatrick.dev). Auto-deploys on push to main.
